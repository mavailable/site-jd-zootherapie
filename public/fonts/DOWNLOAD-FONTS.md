# Polices à télécharger

Télécharge ces polices et place les fichiers .woff2 dans ce dossier :

## Lora (Headings)
- URL : https://gwfh.mranftl.com/fonts/lora?subsets=latin
- Poids : 600, 700
- Format : woff2
- Fichiers attendus : lora-v35-latin-600.woff2, lora-v35-latin-700.woff2

## DM Sans (Body)
- URL : https://gwfh.mranftl.com/fonts/dm-sans?subsets=latin
- Poids : 400, 500, 600
- Format : woff2
- Fichiers attendus : dm-sans-v15-latin-regular.woff2, dm-sans-v15-latin-500.woff2, dm-sans-v15-latin-600.woff2
